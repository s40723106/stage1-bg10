week1

建立協同網站:

1.組長先建立一個網站，步驟與個人網站一致，名稱為Stage1-bg10。

2.組員創立一個空倉儲，之後進入tmp使用使用git clone --recurse 使用git clone --recurse submodules 加入小組倉儲的加入小組倉儲的網址。

3.接著組員進入stage1-bg10執行git remote add後面加上名稱及組員空倉儲的網址。

4.接著就可以git push (需加上remote名稱)。

5.之後組長和組員都可以使用git add 、git commit 、git push (組員需用git push bg10)。
